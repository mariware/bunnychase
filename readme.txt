# BunnyChase Codebase

Welcome to the **BunnyChase** game! Follow the instructions below to ensure proper execution of the program.

## Prerequisites
Before running the game, verify that all required modules and dependencies are installed. Refer to the accompanying documentation for detailed setup instructions.

## Execution Steps
1. Launch `server.py` to initialize the game server.
2. Run `client.py` to start the client application.
3. When prompted by `client.py`, enter the correct server IP address to establish a connection.

## Notes
- This program is designed for local execution and cannot be run on online compilers unless they support the specified dependencies.
- Ensure that all dependencies are properly installed to avoid runtime errors.

Enjoy the game!
